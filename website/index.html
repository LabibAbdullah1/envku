FITUR UTAMA APLIKASI

PHP

PHP SWITCHER

Ubah versi modul PHP di Apache dan PATH Windows CLI secara instan (mendukung versi 8.3 & 8.2).

JS

NODE.JS & NVM INTEGRATION

Pasang dan kelola beberapa versi Node.js secara global dengan bantuan NVM yang terintegrasi penuh.

DB

MYSQL PORTABLE

Instalasi otomatis database MySQL yang terisolasi dengan root akses tanpa password untuk kemudahan lokal.

PMA

PHPMYADMIN

Akses phpMyAdmin versi 5.2.3 terbaru dengan konfigurasi pmadb relasional otomatis dan login bebas error.

DNS

PROJECT WIZARD

Daftarkan virtual host Apache dan domain lokal (.test) secara otomatis langsung melalui aplikasi.

SVC

SERVICE CONTROL

Register dan kendalikan Apache/MySQL langsung sebagai Windows Services secara aman dalam satu dashboard.

RDS

REDIS CACHE

Engine database memori in-memory lokal v5.0 (redis-server.exe) berkinerja tinggi untuk caching dan session.

MP

MAILPIT SANDBOX

Server SMTP lokal dan dashboard webmail logger untuk menangkap dan menguji email pengiriman lokal.

CMP

PHP COMPOSER

Dependency manager PHP (composer.phar) portabel terintegrasi otomatis untuk mengunduh package PHP lokal.

CARA MEMULAI

1

Unduh Aplikasi

Unduh file installer sesuai dengan OS Anda: `.exe` untuk Windows atau `.deb` untuk Linux di bagian unduhan di bawah.

2

Jalankan dengan Hak Akses Tinggi

Di Windows, klik kanan aplikasi lalu pilih Run as Administrator. Di Linux, aplikasi otomatis meminta akses administrator melalui dialog sandi native (PolicyKit/`pkexec`) saat diperlukan.

3

Mulai Unduh Komponen

Buka tab Downloader untuk memasang Apache, PHP, MySQL, Composer, Redis, Mailpit, atau phpMyAdmin ke direktori folder server pilihan Anda secara otomatis.

UNDUH VERSI TERBARU

Pilih versi aplikasi stabil di bawah. Rilis yang diunduh dari aplikasi akan mendapatkan pemberitahuan pembaruan otomatis.
