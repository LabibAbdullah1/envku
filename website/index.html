FITUR UTAMA APLIKASI

PHP

PHP SWITCHER

Ubah versi modul PHP di Apache dan PATH Windows CLI secara instan (mendukung versi 8.3 & 8.2).

JS

NODE.JS & NVM INTEGRATION

Pasang dan kelola beberapa versi Node.js secara global dengan bantuan NVM yang terintegrasi penuh.

DB

MYSQL PORTABLE

Instalasi otomatis database MySQL yang terisolasi dengan root akses tanpa password untuk kemudahan lokal.

PMA

PHPMYADMIN

Akses phpMyAdmin versi 5.2.3 terbaru dengan konfigurasi pmadb relasional otomatis dan login bebas error.

DNS

PROJECT WIZARD

Daftarkan virtual host Apache dan domain lokal (.test) secara otomatis langsung melalui aplikasi.

SVC

SERVICE CONTROL

Register dan kendalikan Apache/MySQL langsung sebagai Windows Services secara aman dalam satu dashboard.

CARA MEMULAI

1

Unduh Aplikasi

Unduh file executable `Envku_1.3.1_x64-setup.exe` terbaru dari bagian unduhan di bawah.

2

Jalankan Sebagai Administrator

Klik kanan aplikasi lalu pilih Run as Administrator agar aplikasi dapat mengedit registry PATH dan berkas Hosts.

3

Mulai Unduh Komponen

Buka tab Downloader untuk memasang Apache, PHP, MySQL, atau phpMyAdmin ke direktori portabel `C:\server` Anda secara otomatis.

UNDUH VERSI TERBARU

Pilih versi aplikasi stabil di bawah. Rilis yang diunduh dari aplikasi akan mendapatkan pemberitahuan pembaruan otomatis.
